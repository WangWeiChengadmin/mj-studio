# 视频生成基本使用

## 创建视频任务

1. 进入 **工作台** 页面
2. 切换到 **视频** 标签
3. 输入提示词描述视频内容
4. 选择模型和参数
5. 点击 **生成**

## 生成参数

不同模型支持不同参数：

| 参数 | 说明 |
|------|------|
| 宽高比 | 视频画面比例 |
| 时长 | 视频长度（秒） |
| 分辨率 | 输出分辨率 |

## 参考图

部分模型支持使用图片作为视频首帧：

1. 点击 **添加参考图**
2. 上传起始帧图片
3. 描述视频动作

## 生成时间

视频生成通常需要 1-5 分钟，这是正常的。

任务会在后台运行，可以离开页面稍后查看结果。

## 视频操作

生成完成后：

- 在线预览播放
- 下载视频文件
- 查看详细信息
- 删除任务
