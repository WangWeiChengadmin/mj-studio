# 支持的视频模型

## 即梦 (Jimeng)

字节跳动的视频生成模型。

**特性**：
- 中文理解好
- 多种分辨率
- 支持图生视频

**参数**：
- 宽高比
- 分辨率

## Veo

Google DeepMind 的视频生成模型。

**特性**：
- 高质量输出
- 长时长视频
- 提示词增强

**参数**：
- 宽高比
- 提示词增强
- 超分辨率

## Sora

OpenAI 的视频生成模型。

**特性**：
- 物理真实感强
- 多种方向选择
- 自定义时长

**参数**：
- 方向（横屏/竖屏/方形）
- 分辨率
- 时长
- 水印

## Grok Video

xAI 的视频生成模型。

**特性**：
- 快速生成
- 支持多种宽高比

**参数**：
- 宽高比

## 模型选择建议

| 场景 | 推荐模型 |
|------|---------|
| 中文内容 | 即梦 |
| 高质量 | Veo、Sora |
| 快速测试 | Grok Video |
