# 绘图基本使用

## 创建绘图任务

1. 进入 **工作台** 页面
2. 在输入框中输入提示词
3. 选择模型
4. 点击 **生成** 按钮

## 提示词技巧

好的提示词应包含：

- **主体**：描述画面的主要内容
- **风格**：如 realistic、anime、oil painting
- **细节**：光影、构图、质量词

### 示例

```
A serene Japanese garden with cherry blossoms, koi pond,
traditional wooden bridge, golden hour lighting,
highly detailed, 8k resolution
```

## 参考图

部分模型支持参考图功能：

1. 点击 **添加参考图**
2. 上传图片
3. 输入提示词描述想要的变化
4. 生成

详见 [参考图功能](./reference-image.md)

## 任务管理

- **进行中**：显示正在生成的任务
- **已完成**：显示生成成功的图片
- **回收站**：已删除的任务，可恢复

## 图片操作

生成完成后，可以：

- 点击查看大图
- 下载图片
- 复制提示词
- 重新生成
- 删除任务
