# 参考图功能

参考图（垫图）功能允许你基于已有图片生成新图片。

## 支持的模型

| 模型 | 参考图支持 |
|------|-----------|
| Midjourney | ✅ |
| Gemini | ✅ |
| Flux | ✅ |
| 豆包 | ✅ |
| GPT-4o | ✅ |
| DALL-E 3 | ❌ |

## 使用方法

1. 在绘图表单中点击 **添加参考图**
2. 选择本地图片上传
3. 在提示词中描述想要的效果
4. 点击生成

## 最佳实践

### 选择合适的参考图

- 图片清晰、主体明确
- 避免过于复杂的场景
- 分辨率不要太低

### 编写提示词

参考图模式下的提示词应描述：
- 想要保留的元素
- 想要改变的部分
- 目标风格

### 示例

**参考图**：一张猫的照片

**提示词**：
```
Transform this cat into a cyberpunk style,
neon lights, futuristic city background,
digital art
```

## 注意事项

- 参考图会被转换为 Base64 上传
- 大图片会增加请求时间
- 建议图片不超过 4MB
