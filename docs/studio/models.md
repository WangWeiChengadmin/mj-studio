# 支持的绘图模型

## Midjourney

最流行的 AI 绘图模型，擅长艺术创作。

**特性**：
- 支持 U1-U4 放大
- 支持 V1-V4 变体
- 支持垫图

**API 格式**：MJ-Proxy

## DALL-E 3

OpenAI 的图像生成模型。

**特性**：
- 精确的文字渲染
- 高质量输出
- 支持多种尺寸

**API 格式**：DALL-E

## Gemini Imagen

Google 的图像生成模型。

**特性**：
- 支持图文混合输入
- 高分辨率输出

**API 格式**：Gemini

## Flux

开源高质量图像生成模型。

**特性**：
- 多种宽高比
- 支持负面提示词
- 高质量细节

**API 格式**：DALL-E (Flux)

## 豆包

字节跳动的图像生成模型。

**特性**：
- 中文理解好
- 支持多种风格

**API 格式**：DALL-E (豆包)

## GPT-4o 图像

通过 GPT-4o 的图像能力生成图片。

**特性**：
- 支持透明背景
- 精确理解复杂提示词

**API 格式**：OpenAI Chat
