# 配置上游

MJ-Studio 本身不提供 AI 服务，需要配置 API 上游（中转站）来使用各项功能。

## 添加上游配置

1. 进入 **设置** → **上游配置**
2. 点击 **添加上游**
3. 填写上游信息：
   - **名称**：自定义名称，便于识别
   - **基础 URL**：API 地址
   - **API 密钥**：从上游获取的密钥
   - **API 格式**：选择对应的格式

## 支持的 API 格式

| 格式 | 说明 | 用途 |
|------|------|------|
| MJ-Proxy | Midjourney 代理格式 | AI 绘图（MJ） |
| Gemini | Google Gemini API | AI 绘图 |
| DALL-E | OpenAI 图像 API | AI 绘图 |
| OpenAI Chat | OpenAI 聊天 API | AI 对话、绘图 |
| Video Unified | 统一视频 API | AI 视频 |

## 配置模型

添加上游后，需要配置可用的模型：

1. 在上游配置中点击 **管理模型**
2. 添加该上游支持的模型
3. 设置模型的类型（绘图/视频/对话）

## 测试配置

配置完成后，可以通过以下方式测试：

- **绘图**：进入工作台，尝试生成图片
- **对话**：进入对话页面，发送消息
- **余额**：在上游配置中查看余额（部分上游支持）
