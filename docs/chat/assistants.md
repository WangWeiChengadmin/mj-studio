# 助手管理

助手是预设的对话角色，可以定制 AI 的行为和回复风格。

## 创建助手

1. 进入 **对话** 页面
2. 在左侧栏点击 **+ 新建助手**
3. 填写助手信息：
   - **名称**：助手的显示名称
   - **描述**：简短描述助手功能
   - **头像**：可选，上传自定义头像
   - **System Prompt**：定义助手的行为

## System Prompt

System Prompt 是给 AI 的指令，决定了助手的人设和行为。

### 示例

**翻译助手**：
```
你是一个专业的翻译助手。用户输入中文时翻译成英文，
输入英文时翻译成中文。只输出翻译结果，不要解释。
```

**代码助手**：
```
你是一个编程专家。回答问题时：
1. 先简要解释原理
2. 给出代码示例
3. 解释关键代码
使用 Markdown 格式化输出。
```

## 使用助手

1. 在对话列表中选择助手
2. 该助手下的所有对话都会使用其 System Prompt
3. 可以随时切换助手

## 开场白建议

助手可以配置开场白，新对话时显示快捷提问按钮。

## 编辑和删除

- 点击助手右侧的编辑按钮修改
- 删除助手会同时删除其下所有对话
