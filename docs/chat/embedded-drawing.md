# 嵌入式绘图

在对话中可以让 AI 直接生成图片，无需切换到工作台。

## 使用方法

在对话中告诉 AI 你想生成图片，例如：

```
帮我画一只可爱的猫咪
```

AI 会返回绘图参数，显示为可点击的绘图卡片。

## 工作原理

1. AI 理解你的绘图需求
2. 生成结构化的绘图参数
3. 前端渲染为绘图组件
4. 点击即可执行绘图

## 支持的参数

- **模型**：选择绘图模型
- **提示词**：生成的提示词
- **参考图**：可选
- **模型参数**：尺寸、质量等

## 最佳实践

- 描述清晰具体的画面
- 可以指定风格、构图
- 可以要求修改参数后再生成

## 示例对话

**用户**：帮我画一个赛博朋克风格的城市夜景

**AI**：
```
[绘图卡片]
模型: midjourney
提示词: Cyberpunk city at night, neon lights...
```

点击卡片即可开始生成。
