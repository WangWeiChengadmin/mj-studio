# 对话基本使用

## 开始对话

1. 进入 **对话** 页面
2. 点击 **新建对话**
3. 在输入框输入消息
4. 按 Enter 或点击发送

## 选择模型

在输入框旁边的下拉菜单中可以切换模型：

- **GPT-4o**：OpenAI 最新模型
- **Claude**：Anthropic 的对话模型
- **DeepSeek**：性价比高的中文模型
- 更多取决于你配置的上游

## 流式输出

消息会以打字机效果逐字显示，提供更好的交互体验。

如需停止生成，点击 **停止** 按钮。

## 多模态输入

支持上传图片与 AI 对话：

1. 点击附件按钮
2. 选择图片
3. 输入相关问题

## 对话管理

- **重命名**：点击对话标题编辑
- **AI 标题**：自动根据内容生成标题
- **压缩**：将长对话压缩为摘要
- **删除**：删除整个对话

## 消息操作

- **复制**：复制消息内容
- **编辑**：修改已发送的消息
- **重新生成**：重新生成 AI 回复
- **删除**：删除单条消息
